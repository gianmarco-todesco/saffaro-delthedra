# saffaro-delthedra
Video to analyse Saffaro's M2 painting

Istruzioni per l'uso:
* Lancia Blender 5.1
* Cancella il cubo iniziale
* Vai nel tab Scripting (dovrebbe essere l'ultimo tab sulla sinistra)
* Nel layout di default ci dovrebbe essere un editor di testo sulla destra. Premi il bottone Open (o usa la voce di menu Text/Open, o usa 7lo shortcut Alt-O). Carica il file M2.py.
* Dai il comando RUN (Alt-P o premi il bottone a forma di triangolo in alto a dx, sopra l'editor di testo)
* Se tutto va bene dovresti vedere comparire l'M2 nella finestra 3D

Al modello è associato un materiale, ma il colore è ancora bianco.
Nel video che ti ho mostrato io avevo assegnato (manualmente) una texture estratta dalla foto del dipinto di Saffaro.
